## Quick example

```typescript
import { Innertube } from 'youtubei.js';

const yt = await Innertube.create();

// Search YouTube
const results = await yt.search('lofi hip hop');
console.log(results.videos);

// Get video info and streaming URL
const info = await yt.getInfo('dQw4w9WgXcQ');
const format = await yt.getStreamingData('dQw4w9WgXcQ', { type: 'audio', quality: 'best' });
console.log(format.url);
```

## Get started

<Steps>
  <Step title="Install the package">
    Install `youtubei.js` from npm, Yarn, or your package manager of choice.
    ```bash
    npm install youtubei.js
    ```
  </Step>
  <Step title="Create an Innertube session">
    Import and initialize the `Innertube` client. Sessions are lightweight and can be reused.
    ```typescript
    import { Innertube } from 'youtubei.js';
    const yt = await Innertube.create();
    ```
  </Step>
  <Step title="Make your first call">
    Call any method on the session to retrieve YouTube data.
    ```typescript
    const info = await yt.getBasicInfo('dQw4w9WgXcQ');
    console.log(info.basic_info.title);
    ```
  </Step>
</Steps>
